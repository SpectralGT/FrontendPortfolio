## 📦 Installation & Setup

To run this portfolio locally, follow these steps:

1. **Clone the repository:**
   ```sh
   git clone https://github.com/SpectralGT/FrontendPortfolio.git
   cd FrontendPortfolio
   ```
2. **Install dependencies:**
   ```sh
   npm install
   ```
3. **Run the development server:**
   ```sh
   npm run dev
   ```
